# Personal Hub

**Personal Hub** is a beginner React project built for learning purposes. It covers the core concepts of React including components, props, state, hooks (`useState`, `useEffect`), controlled inputs, conditional rendering, and working with APIs.

The project includes three widgets:

- **Expense Tracker** — add and delete expenses, calculates a running total
- **Todo List** — add, delete, and toggle todos with visual completion styling
- **Weather Widget** — fetches live weather data using the `wttr.in` API

## Want to build it yourself?

A step-by-step guide is included in the repo — open `GUIDE.html` in your browser to walk through the entire project from scratch.
